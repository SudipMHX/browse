# browse.pro.bd — Project Overview

## Project Idea

`browse.pro.bd` is a **public open-source directory platform** where Bangladeshi developers can showcase:

* Open-source projects
* Portfolios
* Tools
* Libraries
* Indie apps
* SaaS products

**Main Goal:** *"Discover Bangladeshi developers and open-source creations."*

---

## What Type of Platform It Is

A combination of:

```
Awesome List
+
Developer Showcase
+
Product Hunt (lite)
+
Portfolio Gallery
```

Bangladesh-focused, open-source first, curated ecosystem.

---

## Core Philosophy

### 1. Open-Source First
✅ All content GitHub-based  
✅ Anyone can fork, edit, submit PR  
✅ Fully transparent & community-driven

### 2. Static First Architecture
✅ No database initially  
✅ Everything from MDX files + GitHub  
✅ Benefits: Fast, secure, free hosting, amazing SEO

### 3. Curated Ecosystem
✅ Manual review of all submissions  
✅ High quality only, no spam  
✅ Trusted directory

---

## Main Sections

### 1. Projects
Showcase of apps, SaaS, platforms, APIs, AI projects, tools

### 2. Portfolios
Developer portfolios (frontend, backend, designer, student)

### 3. Tools
Free utilities: generators, converters, APIs, CLI tools

### 4. Libraries
