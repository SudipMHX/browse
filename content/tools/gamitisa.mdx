# Gamitisa

Gamitisa.com is a free web platform offering 60+ practical online tools and utilities. It primarily serves users in South Asia (Bangladesh, India, Pakistan), helping students, freelancers, and professionals save time on everyday digital tasks.

## What It Does

- **PDF Tools**: Word to PDF, PDF to Word, PDF to Image, PDF Splitter, PDF Merger, PDF Compressor, PDF Image Extractor, Image to PDF
- **ID Card & Mockup Makers**: Student ID Card, Fake BD NID (Old/Smart), Pakistani CNIC, Facebook/Google ID cards, Birthday Cards
- **Converters & Calculators**: Bangla ↔ Banglish, Age Calculator, BMI Calculator, Color Converter
- **Utilities**: BDIX Server Tester, API Tester, Domain Whois Lookup, CSV/XLS Viewer, Board Results Checker, BD Train Info, Blur Faces, Courier Fraud Checker
- **Localization**: Strong focus on Bengali/Bangla tools and Bangladesh-specific services
